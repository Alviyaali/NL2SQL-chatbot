```
  ├── config.py
  ├── utils.py          → config.py
  ├── validators.py     → config.py
  ├── vanna_setup.py    → config.py, validators.py
  └── seed_memory.py    → config.py, vanna_setup.py
```

## Key Design Notes

| Aspect | Detail |
|--------|--------|
| **Agent singleton** | `vanna_setup.py` creates the Vanna Agent once; all modules share the same instance |
| **Memory singleton** | `DemoAgentMemory` is in-process only — re-seeded on every server start |
| **Config-driven** | All magic numbers live in `config.py`; env vars override defaults |
| **Validation layer** | `validators.py` sits between the LLM and the database — no SQL runs unvalidated |
| **Frontend** | Static HTML/CSS/JS served by FastAPI's `StaticFiles` mount |
